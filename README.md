## Key Features

- **Proxy rotation** - residential and datacenter IPs at scale
- **JS rendering** - headless browser support
- **AI extraction** - structured data from any HTML
- **CI/CD ready** - GitHub Action + Docker + API server
- **Multi-platform** - CLI, Chrome ext, Discord, Telegram
- **Open source** - MIT licensed, free to use
